## 4. DSN 未找到的可能原因

1. **运行时端口驱动**: `cfg_dsn` 是 PCIe IP 的**输入端口**，由 RTL 常量 `64'hA7C3_E5F1_2D8C_49B6` 驱动。该常量在综合时被编码到 LUT 配置中，**不一定**以连续 8 字节形式出现在 .bit 文件中。

2. **FPGA 位流编码**: Xilinx 位流将逻辑配置存储为 LUT 真值表、FF 初值、布线等，64 位常量的比特可能分散在多个配置帧中。

3. **IP 核 DSN 使能**: 需确认 `pcie_7x_0` IP 生成时已启用 DSN Extended Capability，且 `cfg_dsn` 端口已正确连接。
